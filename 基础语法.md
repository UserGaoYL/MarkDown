# 3、代码块

1、代码块
```
int a = 0;
int b = 10;
```

2、指定语言代码块，有高亮效果
```python
int a = 0;
int b = 10;

    "Hello Python"
)
```

3、正文中代码
``int a = 10;  
int b = 10;
``是这么显示的
